## Manual Testing Guide
**Expected Development Version:** `1.3.0-dev.1`
1. Run `npm run start` to boot the application.
2. Open the Settings view from the main interface.
3. Edit the `Default Template` or `Default Path` fields and verify the "saving..." and "saved ✓" indicators appear.
4. Restart the application and confirm the settings persist.
